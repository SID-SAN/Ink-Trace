# Ink-Trace ✍️

A PyTorch-based Convolutional Recurrent Neural Network (CRNN) built from scratch for Optical Character Recognition (OCR) on handwritten and cursive text. 

Ink-Trace is designed to process variable-length handwritten text images, extract geometric features, and decode them into readable strings using Connectionist Temporal Classification (CTC) loss. It is optimized for complex datasets like the IAM Handwriting Database.

## Features

* **Custom CRNN Architecture:** A deep VGG-style Convolutional Neural Network (CNN) paired with a 2-layer Bidirectional LSTM for spatial-to-temporal sequence modeling.
* **Aspect-Ratio Preserving Pipeline:** Smart image preprocessing that scales images to a fixed height (64px) while dynamically padding the width (256px), preventing the "squashing" distortion common in naive resizing.
* **Intelligent Binarization:** Configurable preprocessing filters to erase background noise (like notebook lines) while preserving delicate ink strokes.
* **Dynamic Vocabulary Generation:** Automatically builds a character mapping (`char_map.json`) directly from the training dataset, supporting mixed-case letters and punctuation.
* **Adaptive Training:** Implements PyTorch's `ReduceLROnPlateau` scheduler to dynamically navigate complex loss landscapes and escape CTC "blank bias" plateaus.

## Model Architecture

The network consists of three main components:
1. **Feature Extraction (CNN):** 4 layers of 2D Convolutions (growing from 64 to 256 channels) with `BatchNorm2d`, `ReLU`, and `MaxPool2d` to extract robust visual features from raw pixels.
2. **Sequence Modeling (RNN):** A 2-layer Bidirectional LSTM (Hidden Size: 512, Input Size: 2048) that reads the CNN feature maps as a sequence of time steps to understand the context of connected strokes and ligatures.
3. **Transcription (CTC):** A fully connected linear layer outputs a log-softmax probability distribution over the vocabulary for each time step, which is decoded greedily during inference.

## Installation & Setup

1. **Clone the repository:**
```bash
git clone [https://github.com/SID-SAN/Ink-Trace.git](https://github.com/SID-SAN/Ink-Trace.git)
cd Ink-Trace

```


2. **Set up the environment:**
Ensure you have a modern NVIDIA GPU and CUDA installed. It is recommended to use an Anaconda environment.
```bash
conda create -n torch python=3.10
conda activate torch
pip install torch torchvision torchaudio --index-url [https://download.pytorch.org/whl/cu118](https://download.pytorch.org/whl/cu118)
pip install pandas pyarrow pillow

```



## Usage

### 1. Build Vocabulary

Before training, generate the `char_map.json` mapping from your dataset:

```bash
python vocab.py

```

### 2. Train the Model

Start the training loop. The script automatically uses CUDA if available, clips exploding gradients, and saves weights to the `checkpoints/` directory.

```bash
python train.py

```

### 3. Run Inference

To test the model on a new image, run `predict.py`. The script includes an "X-Ray" diagnostic printout to show the model's uncollapsed frame-by-frame thought process.

```bash
python predict.py

```

*Note: Depending on your input image (digital font vs. handwritten on lined paper), you may need to toggle the binarization threshold `img.point()` inside `predict.py` for optimal stroke clarity.*

## Roadmap & Fine-Tuning
